📝 Resumo Estruturado
O que são LLMs

LLMs (Large Language Models) são modelos de inteligência artificial treinados com grandes volumes de texto para compreender, gerar e manipular linguagem natural.

Eles utilizam arquiteturas de redes neurais profundas e conseguem realizar diversas tarefas como:

geração de texto

tradução

resumo

classificação de informações

Como funcionam

Os LLMs funcionam através de:

1️⃣ Tokenização do texto
2️⃣ Processamento por redes neurais (Transformers)
3️⃣ Previsão probabilística da próxima palavra

Isso permite gerar respostas coerentes baseadas em padrões aprendidos durante o treinamento.

📚 Glossário

LLM (Large Language Model)
Modelo de IA treinado com grandes volumes de texto.

Token
Unidade básica de texto processada pelo modelo.

Prompt
Instrução enviada ao modelo para gerar uma resposta.

Engenharia de Prompts
Técnica de criação de instruções eficientes para obter melhores respostas da IA.

Alucinação da IA
Quando o modelo gera informações incorretas ou inventadas.
